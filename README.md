# 📚 Bookly – Online Book Store

Bookly is a modern and user-friendly online bookstore website. It allows users to explore different categories of books, view details, and add items to their cart or wishlist.

---

## 🚀 Features

- 📖 Browse books by category (Romance, Fiction, Detective, etc.)
- ⭐ Book ratings and reviews
- 💰 Price with discount display
- 🛒 Add to cart functionality
- ❤️ Wishlist feature
- 📱 Fully responsive design
- 🎨 Clean and attractive UI

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Project Goal

This project was created as a university project to demonstrate front-end web development skills and design a simple e-commerce book store interface.
